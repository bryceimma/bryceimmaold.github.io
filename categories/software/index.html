vergelijking workflows Met HUGO maak je zelf een static site aan de hand van layouts. De content vul je op met Markdown bestanden. Enkele voordelen van een static site na dat de layout volledig klaar is: Snel en eenvoudig nieuwe posts maken met markdown files gemakkelijk uploaden via git via config kan je categories en tags maken al je posts in 1 map Enkele nadelen: moeilijk een layout maken moeilijk je layout aanpassen naar jou wens soms moeilijker om te begrijpen Met hugo kan je dus mooie en goede blogs aanmaken en hosten op een geswenst platform.
